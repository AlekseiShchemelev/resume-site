## 🛠️ Как Развернуть Локально

### Шаг 1: Скачайте проект

```bash
# Через Git
git clone https://github.com/alekseishchemelev/resume-website.git
cd resume-website

# Или просто скачайте ZIP-архив и распакуйте
```

### Шаг 2: Подготовьте 3D-модели

Поместите свои STL-файлы в папку `models/`.

> ⚠️ **Важно:** Имена файлов должны точно соответствовать указанным в `index.html` внутри тега `<select>`

### Шаг 3: Запустите локальный сервер

| Вариант                                   | Команда                      | Описание                                                                                                                      |
| ----------------------------------------- | ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| **VS Code + Live Server** (рекомендуется) | —                            | Установите расширение "Live Server", откройте папку проекта, нажмите правой кнопкой на `index.html` → "Open with Live Server" |
| **Python**                                | `python -m http.server 8000` | Откройте в браузере: http://localhost:8000                                                                                    |
| **Node.js**                               | `npx serve`                  | Автоматически запустит сервер                                                                                                 |

## ⚙️ Настройка Персональных Данных

### 📧 Контактная информация

В файле `index.html` найдите блок с классом `contact-links`:

```html
<div class="contact-links">
  <a href="mailto:ВАШ@email.com" class="contact-btn">
    <i class="fas fa-envelope"></i> Email
  </a>
  <a href="tel:+7XXXXXXXXXX" class="contact-btn">
    <i class="fas fa-phone"></i> Телефон
  </a>
  <a href="https://github.com/ВАШ-ПРОФИЛЬ" class="contact-btn">
    <i class="fab fa-github"></i> GitHub
  </a>
  <a href="https://linkedin.com/in/ВАШ-ПРОФИЛЬ" class="contact-btn">
    <i class="fab fa-linkedin"></i> LinkedIn
  </a>
</div>
```

| Поле               | Что заменить                |
| ------------------ | --------------------------- |
| `mailto:`          | Ваш email адрес             |
| `tel:`             | Ваш телефон в формате +7... |
| `github.com/`      | Ссылка на ваш GitHub        |
| `linkedin.com/in/` | Ссылка на ваш LinkedIn      |

### ➕ Добавление новой модели

1. Положите STL-файл в папку `models/`
2. Добавьте опцию в `<select id="stl-select">`:

```html
<option value="models/новая-модель.stl">Название модели</option>
```

3. При необходимости добавьте коррекцию в `script.js`

## 🌐 Публикация на GitHub Pages

### Пошаговая инструкция

1. **Создайте репозиторий на GitHub**

   ```
   https://github.com/ВАШ-ЛОГИН/ИМЯ-РЕПОЗИТОРИЯ
   ```

2. **Загрузите все файлы:**
   - ✅ `index.html`

   - Перейдите в: **Settings → Pages → Source**

   - Folder: `/` (root)

4. **Сохраните и подождите 2-5 минут**

5. **Ваш сайт будет доступен по адресу:**
   ```
   https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ
   ```

## ❓ Возможные Проблемы и Решения

### 🔴 STL-модели не отображаются

| Проверка            | Действие                                |
| ------------------- | --------------------------------------- |
| 📂 Файлы в папке?   | Убедитесь, что файлы лежат в `models/`  |
| 🔤 Имена совпадают? | Проверьте регистр букв (чувствительно!) |
| 🌐 Сервер запущен?  | Откройте через localhost, не `file://`  |
| 📋 Консоль браузера | Нажмите F12 → посмотрите ошибки         |

**Проверка формата STL:**

```bash
# Откройте файл в текстовом редакторе
# Если видите "solid" в начале — это ASCII STL ✅
# Если иероглифы — это бинарный STL ✅
# Оба формата поддерживаются!
```

### 🔴 Модель не по центру или неправильного размера

Откройте `script.js`, найдите функцию `loadSTL()` и измените базовый размер:

```javascript
const scale = maxDim > 10 ? 10 / maxDim : 1;
//                    ↑ измените это число
```

### 🔴 Навигация по якорям не работает

| Причина                 | Решение                                  |
| ----------------------- | ---------------------------------------- |
| Открыто через `file://` | Используйте локальный сервер             |
| Неправильный ID секции  | Проверьте `id="about"` и `href="#about"` |

### 🔴 Анимация появления не срабатывает

Проверьте:

- ✅ Элементы имеют класс `fade-in`
- ✅ Они добавлены в `IntersectionObserver` в `script.js`
- ✅ Браузер поддерживает Intersection Observer

Добавление нового элемента:

```javascript
const fadeElements = document.querySelectorAll(
  ".job, .edu-item, .course-item, .web-app-card, .НОВЫЙ-КЛАСС",
);
```

### 🔴 PDF не генерируется

| Проблема                | Решение                                   |
| ----------------------- | ----------------------------------------- |
| Библиотека не загружена | Проверьте подключение html2pdf.js         |
| Контент не найден       | Убедитесь, что есть `id="resume-content"` |
| CORS ошибки             | Запустите через локальный сервер          |
